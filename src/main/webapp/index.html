JEE7 (CA4) example. Micro Service and REST


Use Firefox and JSON Plugin to display the results
Use Postman to insert data (POST)


No data? Execute the insert script

Swagger Definition :
"http://localhost:8080/microServiceJEE7Ext/api/v1/swagger.json"
"http://localhost:8080/microServiceJEE7Ext/api/v1/swagger.yaml"


--------------REST Methods---------------

initialize a new Sales Order Object with default values
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/initSalesOrder/"

retrieve sales order with id
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrders/1"

retrieve a list of all sales orders
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrders/"

retrieve a list of all sales order TOIs (smaller tranfer objects)
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrderTOIs/"

persist a new sales order (consumes SalesOrder object)
POST
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrders/"

update sales order 1 (consumes SalesOrder object)
PUT
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrders/1"

deletes sales order 1
DELETE
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrders/1"

retrieve a list of all sales order details
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrderDetails/"

retrieve a list of all sales order details for sales order 1
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrderDetailsByOrder/1"


retrieve one sales order detailby id
GET
"http://localhost:8080/microServiceJEE7Ext/api/v1/sample/salesOrderDetails/4"
